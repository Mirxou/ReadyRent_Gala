# Spec: Sovereign Tribunal Dashboard

## 1. Project Context
**Brand:** ReadyRent.Gala (Sovereign Ecosystem)
**Product:** Sovereign Tribunal Dashboard
**Mood:** Cinematic, judicial, high-security, premium "Masterpiece" level.
**Style:** Dark Mode with Aurora Gradients (Amber/Gold/Stone).

## 2. Visual Identity & Tech Strategy
**Strict Color Constraint:** No Blue/Purple. Palette: Earth Tones (Stone #0C0A09), Gold (#CA8A04), Amber (#F59E0B), and Deep Forest Green (#064E3B) for "True Evidence".
**Typography:** "Fraunces" (Heading) paired with "Inter" (Body) for a legal/modern mix.
**Tech:** HTML5, Tailwind CSS v4, Vanilla JS for micro-animations (Framer Motion-like transitions).

## 3. Hero Section Philosophy
**Concept:** "The Gavel of Truth".
**Hero Video:** 1080p Cinematic loop of a slow-motion glass gavel or a high-end judicial hall.
**Imagery:** Geometric patterns representing the "Evidence Vault" (Blockchain/WORM layers).

## 4. Components Structure
1. **Hero (The Adjudicator)**: Massive typography, current "Audit Readiness" score (92%), and live event ticker from the Evidence Log.
2. **The Vault (Evidence Explorer)**: Bento grid showing recent immutable logs, hashes, and integrity checks.
3. **Tribunal Stats**: Real-time charts (BaridiMob success rates, Dispute resolution time).
4. **Sovereign Status**: "Technical Settlement" confirmation.
